# Small-n Management

**Small-n Management** is a short management book about operating when averages mislead, forecasts fail, and every event matters.

## Files

- `manuscript.md` - Markdown manuscript source.
- `book.html` - Print-oriented HTML layout.
- `small-n_management.pdf` - Rendered PDF export.
- `cover.png` - Cover image used by the HTML layout.
